# Практическая работа № 2  
**«Представление двоичных чисел в машинных кодах и выполнение арифметических операций сложения и вычитания двоичных чисел в машинных кодах»**

## Цель и задачи

1. Изучить основные машинные коды (двоичный, восьмеричный, шестнадцатеричный) и способы представления целых чисел.
2. Научиться преобразовывать числа между системами счисления.
3. Выполнить операции сложения и вычитания двоичных чисел, используя правила переноса и вычитания.
4. Развить навыки работы с таблицами истинности и таблицами операций в двоичном коде.

## Теоретическая часть

| Понятие | Определение | Формула / пример |
|---------|-------------|------------------|
| Двоичная система | Система счисления с основанием 2. | \(N = \sum_{i=0}^{k} b_i 2^i\) |
| Оканченный вид | Представление числа с фиксированной длиной (например, 8‑бит). | \(N_{8} = 01010101_2\) |
| Обычное представление | Двоичное представление без ограничений по длине. | \(N = 1101_2\) |
| Запись в 2's complement | Метод хранения отрицательных чисел. | \(-x = 2^n - x\) |
| Перенос (carry) | Факультативный бит, возникающий при сложении. | \(1 + 1 = 10_2\) |
| Битовый сдвиг | Операция, перемещающая биты влево/вправо. | \(1101_2 \ll 1 = 1010_0\) |
| XOR | Операция «исключающего ИЛИ», используется в сложении без переноса. | \(a \oplus b\) |
| AND | Операция «И», используется для вычисления переноса. | \(a \& b\) |
| Таблица истинности | Таблица, показывающая результат операции для всех входов. | |
| Операция сложения | Сложение двух двоичных чисел с учётом переноса. | \((a+b) \mod 2^n\) |
| Операция вычитания | Вычитание двух двоичных чисел с учётом вычитания. | \(a - b = a + \overline{b} + 1\) |

## Ход выполнения работы

1. **Выбор длины машинного кода**  
   - Определите длину слова (например, 8 бит).  
   - Пример: `n = 8`.

2. **Преобразование исходных чисел**  
   - Запишите два целых числа в десятичном виде.  
   - Переведите их в двоичную систему с фиксированной длиной.  
   - *Код (Python)*:
     ```python
     def dec_to_bin(num, bits=8):
         return format(num if num >= 0 else (1 << bits) + num, f'0{bits}b')
     
     a = dec_to_bin(45)   # 00101101
     b = dec_to_bin(27)   # 00011011
     ```

3. **Проверка представления в 2's complement**  
   - Для отрицательных чисел убедитесь, что они корректно закодированы.  
   - *Код*:
     ```python
     neg = dec_to_bin(-13)  # 11110011
     ```

4. **Выполнение сложения**  
   - Сложите два двоичных числа, учитывая перенос.  
   - *Код*:
     ```python
     def add_bin(a, b, bits=8):
         sum_val = int(a, 2) + int(b, 2)
         return format(sum_val & ((1 << bits)-1), f'0{bits}b')
     
     sum_ab = add_bin(a, b)  # 00110100
     ```

5. **Выполнение вычитания**  
   - Вычтите второе число из первого, используя 2's complement.  
   - *Код*:
     ```python
     def sub_bin(a, b, bits=8):
         diff_val = int(a, 2) - int(b, 2)
         return format(diff_val & ((1 << bits)-1), f'0{bits}b')
     
     diff_ab = sub_bin(a, b)  # 00011010
     ```

6. **Проверка результата в десятичном виде**  
   - Переведите полученные двоичные строки обратно в десятичные числа.  
   - *Код*:
     ```python
     def bin_to_dec(bstr):
         return int(bstr, 2) if bstr[0] == '0' else int(bstr, 2) - (1 << len(bstr))
     
     print(bin_to_dec(sum_ab))  # 52
     print(bin_to_dec(diff_ab)) # 18
     ```

7. **Анализ ошибок**  
   - Проверьте наличие переноса при сложении и вычитании.  
   - Оцените, как обрабатываются отрицательные числа.

3. Как преобразовать число `1100` из двоичной в десятичную систему?  
4. Что произойдет при сложении `11111111` и `1` в 8‑битном коде?  
5. Напишите формулу для вычитания `b` из `a` в двоичной системе.  
6. Как определить, является ли двоичное число отрицательным в 2's complement?
