# Практическая работа № 1  
Выполнение перевода чисел из одной системы счисления в другую и арифметических действий над числами в позиционных системах счисления  

## Цель и задачи  
1. Научиться переводить числа между системами счисления с основаниями от 2 до 16.  
2. Освоить выполнение арифметических операций (сложение, вычитание, умножение, деление) над числами в различных позиционных системах.  
3. Развить умения представлять результаты в нужном формате (десятичный, двоичный, шестнадцатеричный).  
4. Приобрести навыки использования встроенных функций языка Python для работы с системами счисления и ручных алгоритмов.

## Теоретическая часть  
| Понятие | Определение | Пример |
|---------|-------------|--------|
| Позиционная система счисления | Система, в которой значение цифры зависит от её позиции. | 101₁₀ = 1·2² + 0·2¹ + 1·2⁰ |
| Основание системы (р) | Количество различных цифр, используемых в системе. | r = 2 (двоичная), r = 16 (шестнадцатеричная) |
| Цифра | Символ, принимающий значение от 0 до r‑1. | 0–9, A–F для r = 16 |
| Перевод из системы r в десятичную | Σ dᵢ·rⁱ, где dᵢ – i‑й цифра справа налево. | 1A₁₆ → 1·16¹ + 10·16⁰ = 26₁₀ |
| Перевод из десятичной в систему r | Последовательное деление на r, остатки записываются в обратном порядке. | 26₁₀ → 1A₁₆ |
| Арифметические операции | Сложение, вычитание, умножение, деление. | 1A₁₆ + 2F₁₆ = 49₁₆ |
| Сдвиг битов | Перемещение битов влево/вправо. | 101₁₀ << 1 = 1010₁₀ |
| Побитовое И, ИЛИ, XOR | Операции над отдельными битами. | 101₁₀ & 011₁₀ = 001₁₀ |

## Ход выполнения работы  

1. **Подготовка среды**  
   - Установить Python 3.10+ (или использовать онлайн‑интерпретатор).  
   - Создать файл `task1.py`.

2. **Функция перевода из системы r в десятичную**  
   ```python
   def to_decimal(num_str: str, base: int) -> int:
       return int(num_str, base)
   ```
   *Пояснение:* Встроенная функция `int` принимает строку и основание, возвращает десятичное целое.  
   *Пример:* `to_decimal('1A', 16)` → `26`.

3. **Функция перевода из десятичной в систему r**  
   ```python
   def from_decimal(num: int, base: int) -> str:
       if num == 0:
           return '0'
       digits = '0123456789ABCDEF'
       res = ''
       while num > 0:
           num, rem = divmod(num, base)
           res = digits[rem] + res
       return res
   ```
   *Пояснение:* Рекурсивный алгоритм деления.  
   *Пример:* `from_decimal(26, 16)` → `'1A'`.

4. **Функция арифметического сложения в произвольной системе**  
   ```python
   def add_in_base(a: str, b: str, base: int) -> str:
       sum_dec = to_decimal(a, base) + to_decimal(b, base)
       return from_decimal(sum_dec, base)
   ```
   *Пример:* `add_in_base('1A', '2F', 16)` → `'49'`.

5. **Функция вычитания**  
   ```python
   def sub_in_base(a: str, b: str, base: int) -> str:
       diff_dec = to_decimal(a, base) - to_decimal(b, base)
       return from_decimal(diff_dec, base)
   ```
   *Пример:* `sub_in_base('50', '1F', 16)` → `'31'`.

6. **Умножение**  
   ```python
   def mul_in_base(a: str, b: str, base: int) -> str:
       prod_dec = to_decimal(a, base) * to_decimal(b, base)
       return from_decimal(prod_dec, base)
   ```
   *Пример:* `mul_in_base('A', '3', 16)` → `'1E'`.

7. **Деление** (целочисленное)  
   ```python
   def div_in_base(a: str, b: str, base: int) -> tuple[str, str]:
       quo = to_decimal(a, base) // to_decimal(b, base)
       rem = to_decimal(a, base) % to_decimal(b, base)
       return from_decimal(quo, base), from_decimal(rem, base)
   ```
   *Пример:* `div_in_base('1A', '3', 16)` → `('5', '1')`.

8. **Проверка работы**  
   - Создать набор тестов (пример: 101₁₀ + 111₁₀, 3A₁₆ * 7₁₆).  
   - Запустить программу и убедиться в корректности результатов.  

9. **Отладка и вывод**  
   - Вывести на экран исходные числа, операции, результаты и их десятичные эквиваленты.  
   ```python
   print(f'{a} + {b} = {res} (decimal: {int(res, base)})')
   ```
