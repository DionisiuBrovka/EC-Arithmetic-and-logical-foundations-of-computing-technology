# Практическая работа № 7. Минимизация логических функций с использованием карт Карно‑Вейча

## Цель и задачи
1. Освоить методику построения и чтения карт Карно‑Вейча (КК‑В) для функций с 3–6 переменными.  
2. Научиться находить минимальные эквивалентные выражения (сумма произведений, произведение сумм).  
3. Развить навыки графического анализа и применения правил объединения групп в КК‑В.  
4. Получить практический опыт работы с таблицами истинности, формулами и их минимизацией.

## Теоретическая часть
1. **Булева функция** – отображение, принимающее логическое значение 0 или 1.  
2. **Таблица истинности** – список всех возможных комбинаций входов и соответствующих выходов.  
3. **Карта Карно‑Вейча (КК‑В)** – двумерная таблица, в которой соседние ячейки отличаются ровно одной переменной.  
4. **Группа** – набор ячеек 1, размерность – степень двойки (1, 2, 4, 8…); каждая группа соответствует минимальному имплиситному выражению.  
5. **Показатель объединения** – переменная, которая меняется внутри группы; она исключается из итогового выражения.  
6. **Идентичность Дизъюнкции**: \(X + X = X\).  
7. **Идентичность Конъюнкции**: \(X \cdot X = X\).  
8. **Правило объединения**: группы могут перекрываться, но каждое 1 должно быть покрыто как минимум одной группой.  
9. **Минимальное SOP (Sum‑of‑Products)** – сумма произведений с минимальным числом переменных.  
10. **Минимальное POS (Product‑of‑Sums)** – произведение сумм с минимальным числом переменных.

## Ход выполнения работы
1. **Составление таблицы истинности**  
   - Запишите все комбинации входных переменных.  
   - Заполните столбец выхода по заданному выражению.  
   - *Пример*:  
     ```text
     A B C | F
     0 0 0 | 1
     0 0 1 | 0
     ...
     ```
2. **Построение карты Карно‑Вейча**  
   - Для 3 переменных создайте таблицу 2x4, для 4 переменных – 4x4, для 5 переменных – 4x8, для 6 переменных – 8x8.  
   - Заполните ячейки согласно таблице истинности.  
   - *Пример для 3 переменных*:  
     ```
     AB\C | 0 | 1
     00   | 1 | 0
     01   | 1 | 1
     11   | 0 | 0
     10   | 1 | 1
     ```
3. **Группировка единиц**  
   - Объедините 1 в группы 2^n, стараясь покрыть как можно больше ячеек.  
   - Учтите, что границы таблицы «переходят» по горизонтали и вертикали.  
   - *Пример*: группа из 4 ячеек (2x2) вокруг (AB=01, C=0/1 и AB=10, C=0/1).  
4. **Вывод минимального выражения**  
   - Для каждой группы запишите произведение переменных, которые **не** меняются внутри группы.  
   - Сложите все произведения.  
   - *Пример*: для группы 4 ячеек, где A меняется, B=1, C=0 → выражение: \(B \cdot \overline{C}\).  
5. **Проверка результата**  
   - Подставьте найденное минимальное выражение в таблицу истинности и убедитесь, что выход совпадает со исходной функцией.  
   - *Пример*:  
     ```text
     A B C | F_min
     0 0 0 | 1
     ...
     ```

## Контрольные вопросы / задания для самопроверки
1. Что такое карта Карно‑Вейча и как она упрощает работу с булевыми функциями?  
2. Как определить, какие ячейки можно объединить в одну группу?  
3. Почему группы должны иметь размерность степени двойки?  
4. Как получить минимальное SOP‑выражение из карт Карно‑Вейча?  
5. Приведите пример функции с 4 переменными, минимизируйте её при помощи КК‑В.  
6. Что происходит, если в КК‑В некоторые 1 не покрыты группами?  
7. Как проверить корректность полученного минимального выражения?  
8. Какие правила объединения применяются при перекрытии групп?  
9. Как изменить карту Карно‑Вейча для функции с 5 переменными?  
10. Сравните минимальные выражения, полученные с помощью КК‑В и метода Крамер‑Эрмитса.
