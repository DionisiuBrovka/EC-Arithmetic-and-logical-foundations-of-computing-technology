# Обязательная контрольная работа № 2

## Цель и задачи контрольной работы

## Краткое теоретическое введение
- **Булева алгебра**: операции И (∧), ИЛИ (∨), НЕ (¬), XOR (⊕), NAND, NOR.  
- **Теорема Де Моргана**: ¬(A∧B)=¬A∨¬B, ¬(A∨B)=¬A∧¬B.  
- **Коды чисел**:  
  - **Бинарный** (двоичный) код.  
  - **Двоично-десятичный код (BCD)**.  
  - **Комплементарные коды**: 1‑комплементарный, 2‑комплементарный.  
- **Математический аппарат**:  
  - **Сумматор**: однобитный (A⊕B⊕C_in).  
  - **Каскадные сумматоры** (два бита, четырёхбитный).  
- **Ассемблер**: основные операнды, команды арифметических операций (ADD, SUB, INC, DEC), работа с регистрами (AX, BX, CX, DX), инструкции перехода (JMP, JZ).  
- **Архитектура ЭВМ**: шина данных, шина адреса, регистры, кэш, кеширование, простейшие схемы ALU.

## Пример решения (задание №2.4)

**Задание**: Перевести число из BCD в двоичный и вывести результат.  

**Код** (x86‑64, NASM синтаксис):

```asm
section .data
    bcd_num db 0x45        ; BCD код 45 (0100 0101)

section .bss
    bin_num resb 1

section .text
    global _start

_start:
    mov al, [bcd_num]      ; загрузить BCD
    mov ah, 0              ; очистить AH

    ; разделяем BCD на два разряда
    mov cl, al
    shr cl, 4              ; старший разряд
    and al, 0x0F           ; младший разряд

    ; перевод в двоичное: старший разряд * 16 + младший разряд
    mov bl, cl
    shl bl, 4
    add bl, al

    mov [bin_num], bl      ; сохраняем результат

    ; завершение программы (Linux sys_exit)
    mov eax, 60
    xor edi, edi
    syscall
```

**Пояснение**:  
- `AL` содержит BCD‑код 45.  
- Сдвиг вправо на 4 бита извлекает старший разряд (4).  
- Маска `0x0F` оставляет младший разряд (5).  
- Умножаем старший разряд на 16 (сдвиг влево) и суммируем с младшим.  
- Получаем двоичное число 45 (0x2D).
