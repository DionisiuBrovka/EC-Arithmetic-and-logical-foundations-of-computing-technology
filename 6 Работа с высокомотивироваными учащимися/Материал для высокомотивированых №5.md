# Арифметические действия над числами с фиксированной запятой  

Числа с фиксированной запятой – это фундаментальная модель представления действительных чисел в цифровой технике, где размерность и точность заранее определены. В современном программном обеспечении они остаются критически важными: от встроенных систем, где требуется экономия памяти, до высокопроизводительных графических и аудио‑процессоров, где фиксированная точность обеспечивает стабильность и предсказуемость вычислений. Понимание их работы открывает доступ к оптимизации алгоритмов, снижению энергопотребления и повышению надёжности систем, что особенно актуально в эпоху Интернета вещей и edge‑компьютинга.

## Цели и задачи  

1. **Понять внутреннюю структуру представления чисел с фиксированной запятой** (разделение на целую и дробную части, масштаб и формат).  
2. **Освоить алгоритмы арифметических операций** (сложение, вычитание, умножение, деление) и их реализацию в аппаратном и программном слоях.  
3. **Развить навыки выбора подходящего формата** (Q‑формат, UQ‑формат, плавающая точность) в зависимости от требований к точности, диапазону и ресурсу.  
4. **Научиться оценивать и минимизировать ошибки округления** и переноса, а также проводить анализ устойчивости алгоритмов.  
5. **Сформировать исследовательское мышление** через анализ современных технологий, сравнительный обзор и разработку мини‑приложений.

## Ключевые понятия  

| Термин | Расширенное определение | Зачем это нужно |
|--------|------------------------|-----------------|
| **Q‑формат** | Представление фиксированной точности в виде `Qm.n`, где `m` – количество битов целой части, `n` – дробной. | Позволяет явно управлять диапазоном и точностью, упрощает конвертацию с плавающей точностью. |
| **Сдвиг с округлением** | Операция смещения разрядов с добавлением компенсации для сохранения точности. | Ключевая операция при умножении и делении, где требуется масштабировать результат. |
| **Перенос (carry/borrow)** | Феномен переноса/выноса при арифметических операциях, который может вызвать ошибку округления. | Важен для корректной реализации арифметики в аппаратном ускорителе. |
| **Каскадный формат** | Способ представления чисел, где каждая часть имеет разный масштаб, объединённый в единую структуру. | Используется в DSP для повышения эффективности обработки разного рода сигналов. |
| **Уровень квантования** | Минимальная изменяемая единица значения в фиксированной точности. | Определяет разрешение сигнала и чувствительность к шуму. |
| **Каскадный умножитель** | Устройство, разбивающее операнды на сегменты для параллельной обработки. | Увеличивает скорость умножения без потери точности. |
| **Параллельный делитель** | Алгоритм деления, использующий несколько итераций и сдвигов для ускорения. | Позволяет быстро выполнять деление в системах реального времени. |
| **Классический формат UQ** | Беззнаковый фиксированный формат `UQn`, где все биты используются для дробной части. | Идеален для представления вероятностей и нормированных значений. |
| **Стабильность алгоритма** | Способность алгоритма сохранять точность при многократных итерациях. | Критично для численных методов, где погрешность может экспоненциально расти. |
| **Точность с плавающей точностью** | Способность фиксированной точности имитировать динамический диапазон плавающей точности при помощи масштабирования. | Позволяет комбинировать преимущества обоих подходов. |

## Примеры и иллюстрации  

```c
// Умножение Q8.8 без 64‑битных типов
int16_t mul_q8_8(int16_t a, int16_t b) {
    int32_t temp = (int32_t)a * (int32_t)b; // 16x16 -> 32
    return (int16_t)(temp >> 8);            // масштабируем обратно
}
```

**Схема деления с использованием сдвига‑вычитания**  

```
 dividend | divisor | shift | quotient | remainder
----------|---------|-------|----------|----------
   0x1F00 |  0x00F0 |   4   |   0x000F |    0x0000
```

**Кейс из индустрии** – *DSP в автомобильных системах*:  
В аудио‑обработке в автомобилях часто используется формат Q1.15 для усиления сигнала. Благодаря фиксированной точности достигается стабильность при низком энергопотреблении, что критично для электромобилей.
