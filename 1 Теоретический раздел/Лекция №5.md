# Системы счисления

## Цели и задачи лекции
1. Понять, как представлять числа в различных системах счисления.  
2. Научиться конвертировать числа между десятичной, двоичной, восьмеричной и шестнадцатеричной системами.  
3. Исследовать применение систем счисления в вычислительной технике и ассемблере.  
4. Разобрать примеры кодирования чисел в различных кодах (двоичный код, код знака, код с дополнением до двух).

## Ключевые понятия
- **База** – основание системы счисления (например, 2, 8, 10, 16).  
- **Двоичная система** – система с основанием 2, использующая цифры 0 и 1.  
- **Октальная система** – система с основанием 8, использующая цифры 0–7.  
- **Десятичная система** – привычная нам система с основанием 10, цифры 0–9.  
- **Шестнадцатеричная система** – система с основанием 16, цифры 0–9 и буквы A–F.  
- **Код знака** – представление знака числа при помощи старшего разряда.  
- **Дополнение до двух** – способ кодирования отрицательных чисел в двоичной системе.  
- **Байт** – 8 бит, основной элемент памяти ЭВМ.  
- **Регистры** – области быстрого доступа в процессоре, хранящие данные и адреса.  
- **Побитовые операции** – операции над отдельными битами (AND, OR, XOR, NOT, сдвиг).

## Основное содержание

### 1. Введение в системы счисления (10 мин)
- История возникновения систем счисления.  
- Почему в компьютерах используют двоичную систему.  
- Связь между разрядностью и диапазоном значений.

### 2. Конвертация чисел (20 мин)
#### 2.1 Десятичное → двоичное  
```
function decToBin(n):
    result = ""
    while n > 0:
        result = (n % 2) + result
        n = n // 2
    return result
```
#### 2.2 Десятичное → октальное  
```
function decToOct(n):
    result = ""
    while n > 0:
        result = (n % 8) + result
        n = n // 8
    return result
```
#### 2.3 Десятичное → шестнадцатеричное  
```
function decToHex(n):
    digits = "0123456789ABCDEF"
    result = ""
    while n > 0:
        result = digits[n % 16] + result
        n = n // 16
    return result
```
#### 2.4 Обратные конвертации  
- Формулы для перевода из любой системы в десятичную.  
- Примеры: `101101₂ → 45₁₀`, `57₈ → 47₁₀`, `2F₁₆ → 47₁₀`.

### 3. Представление целых чисел в памяти (15 мин)
#### 3.1 Код знака  
- Пример: 8‑битный код знака, диапазон – от -128 до 127.  
#### 3.2 Дополнение до двух  
- Как вычислять отрицательные числа: `-x = (~x + 1)`.  
- Пример: `-5` в 8‑битном двоичном коде → `11111011`.  
#### 3.3 Пример в ассемблере  
```asm
MOV AL, 0FFh      ; 255 в 8‑битном коде
NOT AL            ; 0 → 11111111b
INC AL            ; 1 → 11111110b (это -1 в дополнении до двух)
```

### 4. Системы счисления в архитектуре ЭВМ (20 мин)
- Как адреса и данные хранятся в памяти.  
- Влияние разрядности на скорость и ёмкость.  
- Пример: 32‑битный адресный шина → 4 ГБ адресного пространства.  

### 5. Практические задачи (15 мин)
- Перевести число `0x7F` в двоичную и октальную системы.  
- Найти двоичную форму числа `-12` в 16‑битном представлении.  
- Написать простую программу на C, которая выводит число в всех четырёх системах.

## Примеры и иллюстрации
```c
#include <stdio.h>
int main() {
    int n = -12;
    printf("Десятичное: %d\n", n);
    printf("Двоичное (16‑бит): %016b\n", n);
    printf("Октальное: %o\n", n);
    printf("Шестнадцатеричное: %X\n", n);
    return 0;
}
```
