# История развития ЭВМ

## Цели и задачи лекции
- Понять ключевые этапы развития вычислительной техники от механических до современных электронных систем.
- Сопоставить исторические события с появлением фундаментальных архитектурных концепций (модель Тьюринга, von Neumann).
- Изучить влияние теории вычислений и булевой алгебры на эволюцию ЭВМ.
- Научиться выделять основные технологические прорывы и их последствия для программирования.

## Ключевые понятия
| Понятие | Определение |
|---------|-------------|
| Механическая машина | Устройство, использующее рычажные и шестеренчатые механизмы для выполнения вычислений. |
| Электронный вычислительный прибор | Устройство, основанное на электронных компонентах (транзисторах, интегральных схемах). |
| Модель Тьюринга | Абстрактная модель, описывающая алгоритмическую вычислимость через бесконечную ленту и счётчик. |
| Модель von Neumann | Архитектура, где память и процессор разделены, но общедоступны, реализует принцип хранения программы в памяти. |
| Период "первой эпохи" | 1930‑1940 гг.: первые механические и электромеханические вычислители. |
| Период "первой компьютерной эпохи" | 1940‑1950 гг.: появление электронных ламп и первых программируемых ЭВМ. |
| Период интегральных схем | 1960‑1980 гг.: переход к микросхемам, рост мощности и миниатюризации. |
| Период персональных компьютеров | 1980‑2000 гг.: массовое распространение ЭВМ, появление графических интерфейсов. |
| Период суперкомпьютеров | 1990‑наст. время: высокопроизводительные системы, параллелизм, GPU. |

## Основное содержание

### 1. Механические и электромеханические вычислители (30 мин)
- **Альберт Эйнштейн, 1937** – «Механический калькулятор» (появление арифмометра).
- **Машина Перма (1938)** – автоматическая арифметическая машина, использующая шариковые счётные механизмы.
- **Электромеханические счётчики** – первые шаги к автоматизации.

#### Пример: схема работы механической машины
```text
+----+     +----+     +----+
| Ввод| -->| Шестерни| -->| Вывод|
+----+     +----+     +----+
```

### 2. Первая электронная эпоха (40 мин)
- **ENIAC (1945)** – первая полностью электронная программа‑меняемая машина, использующая лампы вакуумные.
- **ЭВМ ЭКВИП (1949)** – применение транзисторов, увеличение надёжности.
- **Модель Тьюринга** – теоретическое обоснование алгоритмической вычислимости.

#### Пример: простая программа на ENIAC (псевдокод)
```text
SET A TO 5
SET B TO 3
ADD A TO B
OUTPUT B   // 8
```

### 3. Архитектура von Neumann и её наследники (30 мин)
- **Архитектура von Neumann** – разделение памяти и процессора, хранение программы в памяти.
- **Машина Manchester (1948)** – первая реализация модели von Neumann.
- **Появление ассемблера** – перевод машинных инструкций в человекочитаемый язык.

#### Пример: ассемблерный код
```assembly
LOAD  R1, #5      ; Загрузка 5 в регистр R1
LOAD  R2, #3      ; Загрузка 3 в регистр R2
ADD   R1, R2      ; R1 = R1 + R2
STORE R1, RESULT  ; Сохранение результата
```

### 4. Интегральные схемы и микропроцессоры (30 мин)
- **Intel 4004 (1971)** – первый коммерческий микропроцессор.
- **Появление RISC** – упрощённые наборы команд.
- **Развитие памяти** – SRAM, DRAM, флэш‑память.

#### Пример: простая инструкция RISC
```assembly
MOV R0, #10      ; Загрузка константы
ADD R0, R1       ; Сложение с содержимым R1
```

### 5. Персональные компьютеры и современные тенденции (30 мин)
- **Apple II, IBM PC** – массовое распространение ЭВМ.
- **Появление графических интерфейсов** – Windows, MacOS.
- **GPU и параллельные вычисления** – ускорение задач ИИ и графики.

#### Пример: CUDA‑псевдокод
```cuda
__global__ void add(int *a, int *b, int *c) {
  int idx = threadIdx.x;
  c[idx] = a[idx] + b[idx];
}
```

## Выводы по лекции
- ЭВМ прошли путь от механических устройств до интегральных микропроцессоров.
- Модели Тьюринга и von Neumann сформировали фундамент вычислительной техники.
- Развитие памяти и процессоров привело к появлению персональных и суперкомпьютеров.
- Современные параллельные архитектуры (GPU, многопроцессорные системы) расширяют возможности вычислений.
