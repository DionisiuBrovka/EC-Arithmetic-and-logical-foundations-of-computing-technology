# Законы и тождества алгебры логики

## Цели и задачи лекции
- Освоить основные законы и тождества булевой алгебры.  
- Понять взаимосвязь между логическими операциями и арифметическими представлениями.  
- Научиться применять тождества для упрощения логических схем и программ на ассемблере.  
- Развить навыки проверки корректности логических выражений.

## Ключевые понятия
| Термин | Определение |
|--------|-------------|
| **Булева переменная** | Переменная, принимающая только два значения: 0 (ложь) и 1 (истина). |
| **Логическая операция** | Операция над булевыми переменными: AND, OR, NOT, XOR и др. |
| **Тождество** | Условие, истинное при всех возможных значениях переменных. |
| **Дистрибутивность** | Свойство распределения одной операции относительно другой. |
| **Двойственное тождество** | Принцип, при котором каждое тождество сохраняет истинность после замены AND↔OR и 0↔1. |
| **Схема логической функции** | Графическое представление логической операции (диаграмма Кентена, таблица истинности). |
| **Кодирование** | Представление логических значений в числовом формате (например, двоичный, шестнадцатеричный). |
| **Ассемблер** | Машинно-ориентированный язык программирования, использующий логические операции. |
| **Условный переход** | Операция, изменяющая поток исполнения программы в зависимости от логического условия. |
| **Мультиплексор** | Устройство, выбирающее один из нескольких входов на основе управляющих сигналов. |

## Основное содержание
### 1. Введение в булеву алгебру (10 мин)
- История возникновения.  
- Обзор базовых операций: `AND`, `OR`, `NOT`, `XOR`.  
- Таблица истинности для каждой операции.

### 2. Основные законы булевой алгебры (20 мин)
- **Идемпотентность**: `A AND A = A`, `A OR A = A`.  
- **Коммутативность**: `A AND B = B AND A`, `A OR B = B OR A`.  
- **Ассоциативность**: `(A AND B) AND C = A AND (B AND C)`.  
- **Дистрибутивность**: `A AND (B OR C) = (A AND B) OR (A AND C)`.  
- **Проведение и отрицание**: `A AND NOT A = 0`, `A OR NOT A = 1`.  

#### Практический пример
```text
A = 1, B = 0
A AND (B OR NOT A) = 1 AND (0 OR 0) = 0
```

### 3. Тождества и их доказательства (25 мин)
- **Тождество де Моргана**: `NOT (A AND B) = NOT A OR NOT B`.  
- **Тождество двойного отрицания**: `NOT (NOT A) = A`.  
- **Тождество разложения**: `A = (A AND B) OR (A AND NOT B)`.  
- **Двойственное тождество**: как получить новое тождество из существующего.

#### Пример доказательства
```text
NOT (A AND B) = NOT A OR NOT B
```
Проверка таблицы истинности (обсуждается в классе).

### 4. Упрощение логических выражений (15 мин)
- Алгоритм минимизации: использование законов для сокращения количества вентилей.  
- Пример: `F = A AND B OR A AND NOT B` упрощается до `F = A`.  

### 5. Применение в ассемблере и кодировании (20 мин)
- Инструкции `AND`, `OR`, `XOR`, `NOT` в ассемблере.  
- Пример программы, реализующей логическую функцию `F = A AND B`.  
```asm
MOV AL, A      ; загрузка A
AND AL, B      ; AL = A AND B
```
- Использование условных переходов (`JZ`, `JNZ`) для реализации логических условий.  

#### Иллюстрация схемы
```text
      +-----+     +-----+
A --->| AND |---->| OUT |
      +-----+     +-----+
```

## Выводы по лекции
- Булева алгебра предоставляет мощный набор законов для формального описания логических операций.  
- Тождества позволяют упрощать и оптимизировать логические схемы и программы.  
- Понимание двойственного тождества расширяет возможности анализа и синтеза логических систем.  
- Практика применения в ассемблере демонстрирует связь теории с реальными задачами в вычислительной технике.
