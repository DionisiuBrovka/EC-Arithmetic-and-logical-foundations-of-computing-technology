# Формы представления чисел

## Цели и задачи лекции  
- Понять основные способы представления чисел в вычислительной технике.  
- Изучить правила преобразования между разными представлениями.  
- Научиться применять представления в практических задачах, связанных с булевой алгеброй и ассемблером.  
- Разобрать примеры кода, демонстрирующие работу с числами в разных кодах.

## Ключевые понятия  
1. **Двоичное представление** – запись чисел в системе счисления с основанием 2.  
2. **Десятичное представление** – привычная для человека система счисления с основанием 10.  
3. **Шестнадцатеричное представление** – система счисления с основанием 16, удобно для работы с байтами.  
4. **Код с фиксированной точкой** – представление дробных чисел, где точка фиксирована на определённом разряде.  
5. **Код с плавающей точкой** – IEEE‑754, разделённый на знак, порядок и мантиссу.  
6. **Двоичный код с дополнением до двух** – способ представления отрицательных целых чисел.  
7. **Двоичный код с пропорциональной кодировкой** – кодирование чисел с плавающей точкой в двоичной системе.  
8. **Байт** – 8 бит, основная единица хранения данных в компьютере.  
9. **Бит** – единица информации, принимающая значения 0 или 1.  
10. **Позиционная система счисления** – система, в которой значение разряда зависит от его позиции.

## Основное содержание  

### 1. Позиционные системы счисления (10 мин)  
- Сравнение десятичной, двоичной и шестнадцатеричной систем.  
- Формула расчёта значения:  
  \[
  N = \sum_{i=0}^{k-1} d_i \cdot b^i
  \]  
  где \(b\) – основание системы, \(d_i\) – разряд.  

#### Пример:  
```text
Десятичное: 156  
Двоичное: 10011100  
Шестнадцатеричное: 9C
```

### 2. Представление целых чисел (10 мин)  
- **Двоичный код с дополнением до двух**  
  - Алгоритм получения отрицательных чисел.  
  - Пример: \(-5\) в 8‑битном виде: `11111011`.  

#### Пример кода (assembly‑подобный):  
```asm
MOV AL, 5          ; 00000101
NEG AL             ; 11111011 (-5)
```

### 3. Представление дробных чисел (10 мин)  
- **Код с фиксированной точкой**  
  - Выбор разрядов для целой и дробной части.  
  - Пример: 8‑битный фиксированный точка Q4.4, число 3.25 → `0011 0100`.  

- **Код с плавающей точкой (IEEE‑754)**  
  - Структура: знак (1 бит), порядок (8 бит), мантисса (23 бита).  
  - Пример: 32‑битный формат: 3.5 → `0100 0011 1100 0000 0000 0000 0000 0000`.  

### 4. Перевод между представлениями (10 мин)  
- Алгоритм преобразования двоичного к шестнадцатеричному: группировка по 4 бита.  
- Пример: `11001101` → `CD`.  
- Перевод с плавающей точкой в двоичную и обратно.  

### 5. Практические задачи (5 мин)  
- Вычисление суммы двух 8‑битных чисел с учётом переполнения.  
- Умножение чисел в формате Q4.4.  
- Пример кода на ассемблере:  

```asm
; Сложение 8‑битных чисел с учётом переполнения
MOV AL, 0x7F      ; 01111111
ADD AL, 0x02      ; 00000010
; AL = 0x81, CF = 0
```

## Примеры и иллюстрации  

```text
Двоичное → Шестнадцатеричное:
1101 0011 1110 0100  →  D3E4
```

```text
Фиксированный точка Q4.4:
Число: 2.75 → 0010 1100
```

```text
IEEE‑754 32‑битный:
3.5  →  0100 0011 1100 0000 0000 0000 0000 0000
```

## Выводы по лекции  
- Числа в вычислительной технике могут храниться в различных форматах, каждый из которых имеет свои преимущества.  
- Понимание позиционных систем и кодов (дополнение до двух, фиксированная точка, плавающая точка) критично для работы с низкоуровневым программированием.  
- Практика преобразования между форматами и работа с ассемблером укрепляет навыки работы с данными на уровне микросхем.
