# Регистры

## Цели и задачи лекции
- Понять роль и назначение регистров в архитектуре ЭВМ.
- Освоить основные типы регистров и их функции.
- Научиться работать с регистрами в ассемблере.
- Оценить влияние регистров на производительность и энергоэффективность системы.

## Ключевые понятия
| Термин | Определение |
|--------|-------------|
| Регистр | Быстрая, временная ячейка памяти внутри процессора. |
| Регистр общего назначения (R) | Хранит данные и адреса, используется в арифметических/логических операциях. |
| Специальный регистр | Регистры, отведённые под управление процессором (например, регистр состояния). |
| Регистр флагов (Flags) | Хранит результаты операций (знки переноса, нуля, знака). |
| Регистр программы (PC) | Указывает на адрес следующей инструкции. |
| Регистр стекового указателя (SP) | Указывает на вершину стека. |
| Регистр сегмента (CS, DS, SS, ES) | Указывает на сегмент памяти в сегментной архитектуре. |
| Регистр счётчика команд (IP) | В некоторых архитектурах используется вместо PC. |
| База и смещение | Концепция адресации с использованием сегмента и смещения. |
| Архитектура CISC vs RISC | Различия в количестве и типах регистров. |

## Основное содержание
### 1. Архитектурные основы регистров (10 мин)
- Позиция регистров в общей схеме ЭВМ.
- Сравнение регистров и памяти (память vs регистры).
- Типы регистров: общие, специальные, сегментные, флаговые.

### 2. Регистр общего назначения (15 мин)
- Обзор наборов регистров в популярных архитектурах (x86, ARM, MIPS).
- Примеры использования: арифметика, логика, обмен данными.
```asm
    MOV R1, #10      ; загрузка литерала в регистр
    ADD R2, R1, R3   ; R2 = R1 + R3
```

### 3. Специальные регистры и их роль (10 мин)
- Регистры состояния, флагов, программы, стека.
- Как они влияют на поток выполнения и управление.
```asm
    CMP R1, R2       ; сравнение, устанавливает флаги
    JZ  label        ; переход при равенстве (зависит от флагов)
```

### 4. Адресация и сегментирование (5 мин)
- Концепция сегмента + смещение.
- Пример: MOV EAX, [DS:EBX+4].
- Как регистры сегментов упрощают доступ к памяти.

### 5. Практическое занятие: работа с регистрами в ассемблере (10 мин)
- Мини‑программа: сложение двух чисел и вывод результата.
- Использование регистров стека для вызова функций.
```asm
    PUSH EBP
    MOV  EBP, ESP
    MOV  EAX, [EBP+8]   ; первый аргумент
    ADD  EAX, [EBP+12]  ; второй аргумент
    POP  EBP
    RET
```
