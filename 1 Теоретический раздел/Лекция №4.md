# Основы теории информации

## Цели и задачи лекции
1. Понять фундаментальные понятия теории информации.  
2. Рассчитать энтропию и взаимную информацию для простых систем.  
3. Связать теоретические показатели с практикой цифровой техники.  
4. Научиться применять формулы энтропии к кодированию данных в ЭВМ.

## Ключевые понятия
- **Информация** – уменьшение неопределённости о событии.  
- **Энтропия** – средняя неопределённость случайной величины.  
- **Кодирование** – преобразование данных в более компактный формат.  
- **Код Хаффмана** – оптимальный код без потерь для дискретных источников.  
- **Код Левенштейна** – код с фиксированной длиной символов.  
- **Сжатие** – уменьшение объёма данных без потерь информации.  
- **Кодирование с потерями** – удаление лишних деталей, допустимое в мультимедиа.  
- **Канал связи** – среда передачи информации, подверженная шуму.  
- **Сигнал** – физическое представление информации.  
- **Шум** – случайные искажения сигнала.

## Основное содержание

### 1. Введение в теорию информации (5 мин)
Постановка задачи измерения информации, история возникновения термина (Клод Шеннон).  

### 2. Энтропия и её свойства (10 мин)
- Формула энтропии \(H(X) = -\sum p_i \log_2 p_i\).  
- Свойства: неотрицательность, максимум для равномерного распределения.  
- Пример расчёта энтропии для двоичного источника.

```text
p(0) = 0.7, p(1) = 0.3
H = -0.7*log2(0.7) - 0.3*log2(0.3) ≈ 0.88 бит
```

### 3. Кодирование без потерь (15 мин)
- **Код Хаффмана**: построение дерева, пример для символов A–D.  
- **Код Левенштейна**: фиксированная длина, простота реализации.  
- Сравнение с теоретическим минимумом энтропии.

```text
Символы: A(0.4), B(0.3), C(0.2), D(0.1)
Код Хаффмана:
A: 0
B: 10
C: 110
D: 111
```

### 4. Кодирование с потерями (10 мин)
- Принцип работы компрессии JPEG, MP3.  
- Понятие «потеря информативности» и его влияние на восприятие.  
- Пример простого кода с усечением.

### 5. Каналы связи и шум (5 мин)
- Модель канала Шеннона–Уитни.  
- Пороговый шум, ошибка передачи.  
- Кодирование для коррекции ошибок (Hamming, Reed–Solomon).

## Примеры и иллюстрации

### Пример 1: Двоичный источник
```text
Событие: бросок монеты
p(H) = 0.5, p(T) = 0.5
H = 1 бит
```

### Пример 2: Код Хаффмана
```text
Символы: {A, B, C, D}
Вероятности: {0.4, 0.3, 0.2, 0.1}
Коды: A=0, B=10, C=110, D=111
Средняя длина = 1.4 бита
```

### Пример 3: Сжатие с потерями
```text
Пиксельное изображение 8×8, каждый канал 8 бит
После компрессии JPEG: 1 бит/пиксель (потеря деталей)
```

## Выводы по лекции
- Энтропия – фундаментальный показатель информационной неопределённости.  
- Кодирование без потерь должно стремиться к энтропии источника.  
- Коды с потерями применяются там, где допустима потеря информации.  
- В каналах связи важно учитывать шум и применять кодирование ошибок.
