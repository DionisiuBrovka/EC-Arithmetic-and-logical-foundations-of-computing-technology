# Кодирование чисел

## Цели и задачи лекции
- Понять основные системы счисления и их применение в вычислительной технике.  
- Изучить правила представления и преобразования чисел в двоичном, восьмеричном и шестнадцатеричном кодах.  
- Познакомиться с двоичным представлением знака и с методами арифметических операций над числами в разных кодах.  
- Научиться использовать примеры преобразований в практических задачах.

## Ключевые понятия
| Термин | Определение |
|--------|-------------|
| **Бинарная система счисления** | Система, использующая только цифры 0 и 1. |
| **Октальная система** | Система с основанием 8 (цифры 0–7). |
| **Шестнадцатеричная система** | Система с основанием 16 (цифры 0–9, A–F). |
| **Двоичное представление знака** | Код, где старший бит указывает на знак числа (0 – положительное, 1 – отрицательное). |
| **Ушедшее (two's complement)** | Представление отрицательных чисел, где отрицательное число получается инвертировав биты и добавив 1. |
| **Преобразование в код** | Перевод числа из одной системы счисления в другую. |
| **Маска** | Набор битов, используемых для выделения части числа. |
| **Сдвиг** | Перемещение битов числа влево или вправо. |
| **Побитовые операции** | Операции, применяемые к отдельным битам (AND, OR, XOR, NOT). |
| **Кодировка** | Процесс преобразования информации в форму, пригодную для хранения или передачи. |

## Основное содержание

### 1. Введение в системы счисления (5 мин)
- История появления двоичной системы в ЭВМ.  
- Сравнение с десятичной системой.  
- Примеры применения в микропроцессорах.

### 2. Представление чисел в двоичной системе (10 мин)
- Формула разложения числа в двоичную систему.  
- Таблица преобразования от 0 до 15.  
- Практическое преобразование: `42 → 101010₂`.

```text
42₁₀ = 32 + 8 + 2 = 2⁵ + 2³ + 2¹ → 101010₂
```

### 3. Перевод между системами счисления (10 мин)
- Двоичное → восьмеричное (по группам по 3 бита).  
- Двоичное → шестнадцатеричное (по группам по 4 бита).  
- Пример: `101010₂ → 52₈ → 2A₁₆`.

```text
101010₂ → 101 010₂ → 5 2₈ → 2A₁₆
```

### 4. Представление знака и отрицательных чисел (10 мин)
- Двоичное представление знака (1‑битовый).  
- Код «двойное дополнение» (two's complement).  
- Пример: `-5` в 8‑битном two's complement.

```text
5  = 00000101₂
~5 = 11111010₂
-5 = 11111010₂ + 1 = 11111011₂
```

### 5. Арифметика над двоичными числами (10 мин)
- Сложение, вычитание, умножение, деление в двоичной системе.  
- Пример сложения: `1101₂ + 1011₂`.

```text
  1101
+ 1011
------
 11000  (с переносом)
```

### 6. Маски и побитовые операции (10 мин)
- Создание масок (AND, OR).  
- Выделение младшего байта: `num & 0xFF`.  
- Сдвиги: `num << 2`, `num >> 3`.  
- Практический пример: извлечение 4‑битного поля из 8‑битного числа.

```text
num = 0b11010101
mask = 0b00001111
field = (num & mask) >> 0  // 0101₂ → 5₁₀
```

## Примеры и иллюстрации

- **Преобразование числа 255₁₀** в разные коды:

```text
255₁₀ = 11111111₂ = 377₈ = FF₁₆
```

- **Вычисление 12₁₀ + (-5)₁₀** в 8‑битном two's complement:

```text
12  = 00001100₂
-5  = 11111011₂
Sum = 11100111₂ = -7₁₀
```

## Выводы по лекции
- Системы счисления с разными основаниями играют ключевую роль в цифровой обработке данных.  
- Двоичная система является фундаментом любой современной вычислительной техники.  
- Понимание представления знака и двухкомплементарного кода позволяет корректно работать с отрицательными числами в машинном коде.  
- Побитовые операции и маски являются мощным инструментом при работе с регистрами и памятью.
